1. Input: It is the process of providing data and instructions to the computer. Input devices are used to enter data and instructions in the computer system.Some example of input devices are keyboard,mouse,scanner,and microphone.

  2. Process: It is the process of executing or calculating the data as per the instructions provided by the user. It is done by CPU (Central Processing Device).

  3. Output:The result after processing is called output.The output is provided to the user through output devices. Some example of output devices are monitor,speaker and printer.

  4. Storage/Memory:The computer can store data either temporarily or permanentally. Memory devices are used to store data. Some examples of memory devices are RAM,ROM,Hard disk,CD and DVD.

  5. Feedback:Feedback of the output can be used to update/modify the inputs for producing the desired output.
